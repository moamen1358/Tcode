# Tessera Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** A borderless, keyboard-driven Linux GUI app that tiles N terminal panes in a balanced grid — pick a number, get that many shells.

**Architecture:** Cargo workspace. `tessera-core` holds all pure logic (grid geometry + config), unit-tested with no GTK dependency. `tessera` is a thin GTK4 binary that renders panes with VTE terminal widgets, styled via CSS, driven by Alt-key shortcuts.

**Tech Stack:** Rust (stable 1.96), `gtk4` (gtk4-rs) + `vte4` bindings, `serde` + `toml` for config. System libs: GTK 4.14, VTE 0.76 on Pop!_OS 24.04 (Wayland).

## Global Constraints

- **Platform:** Linux only. System GTK **4.14.5**, VTE **0.76.0**. Do not enable gtk4 features above `v4_14`.
- **GTK/VTE API source of truth:** exact `gtk4`/`vte4` crate versions and the `vte4::Terminal::spawn_async` call come from the `verify-gtk4-vte4-api` workflow output, and are **confirmed by `cargo build`** before any code depends on them. The compiler wins over any snippet in this plan.
- **Keybindings:** `Alt+h/j/k/l` focus, `Alt+z` zoom toggle, `Alt+r` restart pane, `Alt+1..9` re-grid, `Alt+q` quit.
- **Pane count range:** 1–16 (CLI arg); picker + `Alt+digit` cover 1–9.
- **Quality gate per task:** `cargo test -p tessera-core` (core tasks) and/or `cargo build` + `cargo clippy -- -D warnings` (all) must pass.
- **Commits:** one commit per completed task. Per the user's global rule, ask for blanket commit authorization once before executing; if declined, work stays uncommitted until requested.
- **Zero-config:** the app runs with no config file; all settings have defaults.

/// Number of panes in each row of a balanced `n`-pane grid.
/// `rows = round(sqrt(n))`; the first `n % rows` rows get one extra pane.
pub fn layout(n: usize) -> Vec<usize> {
    if n == 0 {
        return Vec::new();
    }
    let rows = ((n as f64).sqrt().round() as usize).max(1);
    let base = n / rows;
    let extra = n % rows;
    (0..rows)
        .map(|r| if r < extra { base + 1 } else { base })
        .collect()
}
```

#[derive(Clone, Copy, Debug, PartialEq, Eq)]
pub enum Dir {
    Left,
    Right,
    Up,
    Down,
}

/// Move focus within a jagged grid described by `widths` (panes per row).
/// Clamps at edges; up/down clamp the column into the target row.
pub fn neighbor(widths: &[usize], row: usize, col: usize, dir: Dir) -> (usize, usize) {
    if widths.is_empty() {
        return (0, 0);
    }
    let row = row.min(widths.len() - 1);
    let col = col.min(widths[row].saturating_sub(1));
    match dir {
        Dir::Left => (row, col.saturating_sub(1)),
        Dir::Right => (row, (col + 1).min(widths[row] - 1)),
        Dir::Up => {
            let nr = row.saturating_sub(1);
            (nr, col.min(widths[nr] - 1))
        }
        Dir::Down => {
            let nr = (row + 1).min(widths.len() - 1);
            (nr, col.min(widths[nr] - 1))
        }
    }
}

/// Flat (row-major) index of the pane at `(row, col)`.
pub fn flat_index(widths: &[usize], row: usize, col: usize) -> usize {
    widths[..row].iter().sum::<usize>() + col
}
```

#[derive(Debug, Clone, Deserialize)]
#[serde(default)]
pub struct Config {
    pub font: String,
    pub font_size: u32,
    pub gap: u32,
    pub startup_command: String,
    pub theme: Theme,
}

#[derive(Debug, Clone, Deserialize)]
#[serde(default)]
pub struct Theme {
    pub background: String,
    pub foreground: String,
    pub accent: String,
    pub palette: Vec<String>,
}

const DEFAULT_PALETTE: [&str; 16] = [
    "#45475a", "#f38ba8", "#a6e3a1", "#f9e2af", "#89b4fa", "#f5c2e7", "#94e2d5", "#bac2de",
    "#585b70", "#f38ba8", "#a6e3a1", "#f9e2af", "#89b4fa", "#f5c2e7", "#94e2d5", "#a6adc8",
];

impl Default for Config {
    fn default() -> Self {
        Config {
            font: "JetBrains Mono".into(),
            font_size: 11,
            gap: 8,
            startup_command: String::new(),
            theme: Theme::default(),
        }
    }
}

impl Default for Theme {
    fn default() -> Self {
        Theme {
            background: "#1e1e2e".into(),
            foreground: "#cdd6f4".into(),
            accent: "#89b4fa".into(),
            palette: DEFAULT_PALETTE.iter().map(|s| s.to_string()).collect(),
        }
    }
}

impl Config {
    pub fn load() -> Self {
        Self::from_path(&config_path())
    }

    pub fn from_path(path: &Path) -> Self {
        match std::fs::read_to_string(path) {
            Ok(s) => Self::from_str_or_default(&s),
            Err(_) => Config::default(),
        }
    }

    pub fn from_str_or_default(s: &str) -> Config {
        match toml::from_str(s) {
            Ok(c) => c,
            Err(e) => {
                eprintln!("tessera: config parse error ({e}); using defaults");
                Config::default()
            }
        }
    }
}

fn config_path() -> PathBuf {
    let base = std::env::var_os("XDG_CONFIG_HOME")
        .map(PathBuf::from)
        .unwrap_or_else(|| {
            let home = std::env::var_os("HOME").map(PathBuf::from).unwrap_or_default();
            home.join(".config")
        });
    base.join("tessera").join("config.toml")
}

use gtk4::prelude::*;
use gtk4::{CssProvider, STYLE_PROVIDER_PRIORITY_APPLICATION};

pub fn rgba(hex: &str) -> RGBA {
    RGBA::parse(hex).unwrap_or_else(|_| RGBA::new(0.0, 0.0, 0.0, 1.0))
}

pub fn install_css(gap: u32, accent: &str, bg: &str) {
    let css = format!(
        ".grid-root {{ background-color: {bg}; padding: {gap}px; }}
         box.row {{ }}
         .pane {{ border-radius: 8px; border: 2px solid transparent; }}
         .pane.active {{ border: 2px solid {accent}; }}",
    );

    provider.load_from_string(&css);

        gtk4::style_context_add_provider_for_display(
            &display,
            &provider,
            STYLE_PROVIDER_PRIORITY_APPLICATION,
        );
    }

- [ ] **Step 1: Implement `pane.rs`**

```rust
use gtk4::prelude::*;
use gtk4::{glib, Label, Overlay};
use tessera_core::config::Config;
use vte4::prelude::*;
use vte4::{PtyFlags, Terminal};

use crate::theme::rgba;

pub struct Pane {
    pub root: Overlay,
    pub terminal: Terminal,
}

impl Pane {
    pub fn new(cfg: &Config) -> Pane {
        let terminal = Terminal::new();
        terminal.add_css_class("pane");

        // Colors via the VTE API (CSS cannot reach cell colors).
        let fg = rgba(&cfg.theme.foreground);
        let bg = rgba(&cfg.theme.background);
        let palette: Vec<_> = cfg.theme.palette.iter().map(|c| rgba(c)).collect();
        let palette_refs: Vec<&_> = palette.iter().collect();
        terminal.set_colors(Some(&fg), Some(&bg), &palette_refs);

        if let Ok(font) = gtk4::pango::FontDescription::from_string(&format!(
            "{} {}",
            cfg.font, cfg.font_size
        ))
        .into()
        {
            terminal.set_font(Some(&font));
        }

        let root = Overlay::new();
        root.set_child(Some(&terminal));

        let pane = Pane { root, terminal };
        pane.spawn(cfg);
        pane
    }

    fn spawn(&self, cfg: &Config) {
        let shell = std::env::var("SHELL").unwrap_or_else(|_| "/bin/sh".into());
        let cwd = std::env::current_dir().ok();
        let argv: Vec<String> = if cfg.startup_command.trim().is_empty() {
            vec![shell.clone()]
        } else {
            vec![
                shell.clone(),
                "-c".into(),
                format!("{}; exec {}", cfg.startup_command, shell),
            ]
        };
        let argv_ref: Vec<&str> = argv.iter().map(|s| s.as_str()).collect();

        // spawn_async signature finalized against vte4 docs + compiler.
        self.terminal.spawn_async(
            PtyFlags::DEFAULT,
            cwd.as_deref().and_then(|p| p.to_str()),
            &argv_ref,
            &[],
            glib::SpawnFlags::DEFAULT,
            || {},
            -1,
            gtk4::gio::Cancellable::NONE,
            move |_res| {},
        );

        let term = self.terminal.clone();
        let overlay = self.root.clone();
        self.terminal.connect_child_exited(move |_t, _status| {
            let label = Label::new(Some("[exited — Alt+r to restart]"));
            label.add_css_class("exited");
            label.set_halign(gtk4::Align::Center);
            label.set_valign(gtk4::Align::Center);
            overlay.add_overlay(&label);
            let _ = &term;
        });
    }

    pub fn respawn(&self, cfg: &Config) {
        // Clear any exit overlay children, then spawn again.
        while let Some(child) = self.root.last_child() {
            if child.downcast_ref::<Terminal>().is_some() {
                break;
            }
            self.root.remove_overlay(&child);
        }
        self.spawn(cfg);
    }

    pub fn grab_focus(&self) {
        self.terminal.grab_focus();
    }

    pub fn set_active(&self, active: bool) {
        if active {
            self.terminal.add_css_class("active");
        } else {
            self.terminal.remove_css_class("active");
        }
    }
}
```

```rust
use gtk4::prelude::*;
use gtk4::{Box as GtkBox, Orientation};
use tessera_core::config::Config;
use tessera_core::grid::{flat_index, layout, neighbor, Dir};

use crate::pane::Pane;

pub struct Grid {
    pub root: GtkBox,
    panes: Vec<Pane>,
    rows: Vec<GtkBox>,
    widths: Vec<usize>,
    focus: (usize, usize),
    zoomed: bool,
}

impl Grid {
    pub fn new(n: usize, cfg: &Config) -> Grid {
        let widths = layout(n.clamp(1, 16));
        let gap = cfg.gap as i32;

        let root = GtkBox::new(Orientation::Vertical, gap);
        root.add_css_class("grid-root");
        root.set_homogeneous(true);

        let mut panes = Vec::new();
        let mut rows = Vec::new();
        for &w in &widths {
            let row = GtkBox::new(Orientation::Horizontal, gap);
            row.add_css_class("row");
            row.set_homogeneous(true);
            for _ in 0..w {
                let pane = Pane::new(cfg);
                pane.root.set_hexpand(true);
                pane.root.set_vexpand(true);
                row.append(&pane.root);
                panes.push(pane);
            }
            root.append(&row);
            rows.push(row);
        }

        let mut grid = Grid {
            root,
            panes,
            rows,
            widths,
            focus: (0, 0),
            zoomed: false,
        };
        grid.update_active();
        grid
    }

    fn focused_index(&self) -> usize {
        flat_index(&self.widths, self.focus.0, self.focus.1)
    }

    pub fn focused_pane(&self) -> &Pane {
        &self.panes[self.focused_index()]
    }

    fn update_active(&self) {
        let active = self.focused_index();
        for (i, p) in self.panes.iter().enumerate() {
            p.set_active(i == active);
        }
        self.panes[active].grab_focus();
    }

    pub fn move_focus(&mut self, dir: Dir) {
        if self.zoomed {
            return;
        }
        self.focus = neighbor(&self.widths, self.focus.0, self.focus.1, dir);
        self.update_active();
    }

    pub fn toggle_zoom(&mut self) {
        self.zoomed = !self.zoomed;
        let (fr, fc) = self.focus;
        for (r, row) in self.rows.iter().enumerate() {
            row.set_visible(!self.zoomed || r == fr);
        }
        // hide non-focused panes inside the focused row
        let mut idx = 0;
        for (r, &w) in self.widths.iter().enumerate() {
            for c in 0..w {
                let visible = !self.zoomed || (r == fr && c == fc);
                self.panes[idx].root.set_visible(visible);
                idx += 1;
            }
        }
        self.focused_pane().grab_focus();
    }

    pub fn restart_focused(&self, cfg: &Config) {
        self.focused_pane().respawn(cfg);
    }
}
```

use gtk4::{Application, ApplicationWindow};
use std::cell::RefCell;
use std::rc::Rc;
use tessera_core::config::Config;

use crate::grid::Grid;
use crate::{keys, picker, theme};

    pub window: ApplicationWindow,
    pub cfg: Config,
    pub grid: Option<Grid>,
}

pub type Shared = Rc<RefCell<State>>;

pub fn build(app: &Application, preset: Option<usize>) {
    let cfg = Config::load();
    theme::install_css(cfg.gap, &cfg.theme.accent, &cfg.theme.background);

    let window = ApplicationWindow::builder()
        .application(app)
        .decorated(false)
        .title("Tessera")
        .build();
    window.maximize();

    let state: Shared = Rc::new(RefCell::new(State {
        window: window.clone(),
        cfg,
        grid: None,
    }));

    keys::install(&window, &state);

    match preset {
        Some(n) => show_grid(&state, n),
        None => show_picker(&state),
    }
    window.present();
}

pub fn show_picker(state: &Shared) {
    let st = state.clone();
    let widget = picker::build(move |n| show_grid(&st, n));
    state.borrow().window.set_child(Some(&widget));
}

pub fn show_grid(state: &Shared, n: usize) {
    let grid = {
        let s = state.borrow();
        Grid::new(n, &s.cfg)
    };
    state.borrow().window.set_child(Some(&grid.root));
    state.borrow_mut().grid = Some(grid);
}

```rust
use gtk4::gdk::{Key, ModifierType};
use gtk4::prelude::*;
use gtk4::{glib, ApplicationWindow, EventControllerKey};
use tessera_core::grid::Dir;

use crate::app::{show_grid, Shared};

pub fn install(window: &ApplicationWindow, state: &Shared) {
    let controller = EventControllerKey::new();
    let st = state.clone();
    controller.connect_key_pressed(move |_c, keyval, _code, modifiers| {
        if !modifiers.contains(ModifierType::ALT_MASK) {
            return glib::Propagation::Proceed;
        }
        // re-grid on Alt+digit
        if let Some(d) = keyval.to_unicode().and_then(|c| c.to_digit(10)) {
            if (1..=9).contains(&d) {
                show_grid(&st, d as usize);
                return glib::Propagation::Stop;
            }
        }
        let mut s = st.borrow_mut();
        let handled = match keyval {
            Key::h => s.grid.as_mut().map(|g| g.move_focus(Dir::Left)).is_some(),
            Key::j => s.grid.as_mut().map(|g| g.move_focus(Dir::Down)).is_some(),
            Key::k => s.grid.as_mut().map(|g| g.move_focus(Dir::Up)).is_some(),
            Key::l => s.grid.as_mut().map(|g| g.move_focus(Dir::Right)).is_some(),
            Key::z => s.grid.as_mut().map(|g| g.toggle_zoom()).is_some(),
            Key::r => {
                if let Some(g) = s.grid.as_ref() {
                    g.restart_focused(&s.cfg);
                }
                s.grid.is_some()
            }
            Key::q => {
                s.window.close();
                true
            }
            _ => false,
        };
        if handled {
            glib::Propagation::Stop
        } else {
            glib::Propagation::Proceed
        }
    });
    window.add_controller(controller);
}
```
